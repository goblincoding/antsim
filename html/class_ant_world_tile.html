Detailed Description

This class represents the sim's 2D world squares (tiles) and is designed to be independent of any specific graphics framework.

As an abstract base class, users must therefore ensure that they inherit from it and implement

See also:
updateGraphics in order to apply graphical changes when the tile type is changed.
